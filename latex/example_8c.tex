This function is an implementation of the \mbox{\hyperlink{showworld_8h_aa7881dde0e99af6e6576a7401f3c4586}{get\+\_\+agent\+\_\+info\+\_\+at()}} function definition. It only works for \mbox{\hyperlink{struct_a_g_e_n_t}{A\+G\+E\+NT}} and \mbox{\hyperlink{struct_w_o_r_l_d}{W\+O\+R\+LD}} structures defined in this example.

It basically receives a pointer to a \mbox{\hyperlink{struct_w_o_r_l_d}{W\+O\+R\+LD}} structure, obtains the \mbox{\hyperlink{struct_a_g_e_n_t}{A\+G\+E\+NT}} structure in the given coordinates, and returns the agent information in a bit-\/packed {\ttfamily unsigned int}.

\begin{DoxyNote}{Note}
This is an example which will probably not work in a fully functional game. Students should develop their own implementation of \mbox{\hyperlink{showworld_8h_aa7881dde0e99af6e6576a7401f3c4586}{get\+\_\+agent\+\_\+info\+\_\+at()}} and agent/world data structures.
\end{DoxyNote}
